## Features

### UI / Design

- **Dark luxury editorial** aesthetic — deep black background, gold accents, serif display fonts
- Custom animated cursor (hides on mobile)
- Page loader with animated progress bar
- Ken Burns hero slideshow with dot navigation
- Marquee trust strip
- Staggered scroll reveal animations
- Floating badge, image stack, and hover effects

### Functionality

- **Live destination search** with autocomplete suggestions
- **Filter tabs** — filter destinations by type (Beaches, Mountains, Culture, Wildlife)
- **Load more** destinations dynamically
- **Wishlist toggle** on destination cards (heart icon)
- **Date pickers** with min-date validation
- **Booking modal** with full form validation (required fields, email format)
- **Newsletter form** with email validation
- **Testimonial carousel** with auto-advance and manual controls
- **Animated counters** (stats section, triggers on scroll)
- **Toast notifications** for user feedback
- **Smooth scroll** for all anchor links
- **Hamburger mobile menu** with body-scroll lock

### Responsive Breakpoints

- Desktop: 1240px max-width grid
- Tablet (≤ 1024px): 2-col services, stacked discover section
- Mobile (≤ 768px): single column, stacked forms, hidden cursor
- Small mobile (≤ 480px): compact spacing throughout

## Tech Stack

- **HTML5** — semantic markup, ARIA labels
- **CSS3** — custom properties, Grid, Flexbox, keyframe animations, clamp(), IntersectionObserver
- **Vanilla JavaScript (ES6+)** — no dependencies
- **Google Fonts** — Playfair Display, Outfit
- **Images** — Unsplash (free to use)

## Quick Start

Simply open `index.html` in a browser. No build step required.
For best experience, serve over a local server (e.g. VS Code Live Server).
